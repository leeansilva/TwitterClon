## REQUIRED
Local server (xampp, wampserver)

## Installation

Download my repository, and copy it to

```bash
C:\XAMPP\htdocs\Twitter
```

Or where you have your "main" of your server.

## RUN

Open the folder "log in", register, and that's it.


## For a new registration

To register again, delete the existing database(F12 + Application/ IndexedDB + delete)

If there is any bug let me know(send me an email), this is the beta version.
